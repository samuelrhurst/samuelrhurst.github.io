# Sam Hurst's Blog

https://samuelrhurst.github.io/

## Hugo Static Site Generator

This blog uses [Hugo](https://github.com/gohugoio/hugo) (a popular open-source static site generator written in Go), to build the markdown posts placed in the `/content` dir into a static html website, ready to be served.

## Hextra Theme for Hugo

This blog uses the [Hextra](https://imfing.github.io/hextra/) theme for Hugo to style the final website.

## GitHub Actions Deployment to GitHub Pages

This repo use a GitHub Actions workflow ([`.github/workflows/pages.yaml`](./.github/workflows/pages.yaml)) to publishing the blog to [GitHub Pages](https://github.blog/changelog/2022-07-27-github-pages-custom-github-actions-workflows-beta/).
